# Post Category Filter

A lightweight WordPress plugin that adds an instant search box to your post categories and custom taxonomy lists in the admin area. This makes it easy to quickly find and filter terms, especially in large or complex taxonomies.

Originally created by **Javier Villanueva**, and now officially adopted and maintained by **Ivijan Stefan Stipic**.
